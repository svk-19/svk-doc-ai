# SVK Doc AI

Professional AI-powered Multi-Document Assistant built using RAG, Gemini AI, ChromaDB and Streamlit.

## Features

- PDF Upload
- DOCX Upload
- Semantic Search
- Context Retrieval
- AI Question Answering
- Chat Memory
- Document Summarization
- Export Chat
- Modern Dark UI

## Tech Stack

- Python
- Streamlit
- Gemini 2.5 Flash
- ChromaDB
- Sentence Transformers
- LangChain Text Splitters

## Project Architecture

Document Upload
↓
Text Extraction
↓
Chunking
↓
Embeddings
↓
ChromaDB Storage
↓
Semantic Retrieval
↓
Gemini AI
↓
Answer Generation

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
